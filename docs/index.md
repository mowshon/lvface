# lvface

`lvface` detects and aligns faces, produces 512-dimensional LVFace embeddings, and compares
people across portraits and group photos.

See the [project README](https://github.com/mowshon/lvface#readme) for installation, examples,
model provenance, licensing notes, and biometric-data safety guidance.

The default `0.35` cosine threshold and percentage mapping are provisional starting points, not
probabilities or domain-general operating thresholds. Calibrate them on representative,
properly licensed data before deployment.
